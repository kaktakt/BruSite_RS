Описание

Это первый в истории Женский сервер созданный в 2014 году, который стабильно работает и развивается по сей день, и в след за собой породил очень большое количество похожих женских серверов. Проект, который изобрел самое большое количество модных трендов и игровых решений всему игровому сообществу серверов по Counter-Strike 1.6, начиная с 2014 года и по сей день. Проект, который был бессменным лидером по онлайну игроков по версии мирового рейтинга gametracker
